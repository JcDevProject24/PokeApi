# Changelog

Todos los cambios relevantes de este proyecto están documentados en este archivo.

El formato sigue [Keep a Changelog](https://keepachangelog.com/es/1.1.0/)
y este proyecto usa [versionado semántico](https://semver.org/lang/es/).

---

## [1.2.0] - 2026-03-10

### Añadido
- Favicon SVG de pokébola (`assets/favicon.svg`)
- `manifest.json` para soporte PWA (instalable desde móvil/escritorio)
- Meta tags SEO: `description`, `author`, `theme-color`
- Meta tags Open Graph para previsualización en redes sociales

## [1.0.0] - 2026-02-21

### Añadido
- Catálogo de Pokémon con scroll infinito (25 por página)
- Búsqueda en tiempo real con autocompletado (hasta 6 sugerencias)
- Filtro por tipo con nombres en español
- Filtro por Pokédex regional
- Filtro por generación (ordenados por ID)
- Botón de reset de filtros
- Modal de detalle con imagen oficial, tipos, altura y peso
- Sección de habilidades con indicador de habilidad oculta (★)
- Barras de estadísticas base animadas
- Mapa interactivo del hábitat usando Leaflet + OpenStreetMap
- Cierre del modal con botón, clic en fondo o tecla Escape
- Mensajes de error/info con desaparición automática (3 s)
- Cruce de datos entre `/pokemon` y `/pokemon-species`
- Diseño responsivo para móvil, tablet y escritorio
